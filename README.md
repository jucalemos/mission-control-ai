# mission-control-ai

Projeto desenvolvido para a Global Solution 2026.1 da disciplina de Pensamento Computacional e Automação com Python.

## Sobre o projeto

O Mission Control AI é um sistema que simula o monitoramento de uma missão espacial. A partir de dados como temperatura, comunicação, bateria, oxigênio e estabilidade, o programa analisa cada ciclo da missão, identifica riscos e gera relatórios para auxiliar na tomada de decisões.

## Funcionalidades

- Análise dos dados da missão;
- Classificação dos ciclos em Normal, Atenção ou Crítico;
- Cálculo do nível de risco;
- Identificação da tendência da missão;
- Identificação da área mais afetada;
- Geração de recomendações automáticas;
- Relatório final da missão.
